# DiaryAI

Зашифрованный дневник с end-to-end шифрованием. Открытый код.

## Концепция

- **E2E шифрование**: пароль никогда не уходит на сервер. Сервер хранит только зашифрованные блобы.
- **Local-first**: приложение работает офлайн, синхронизация — опциональная.
- **Кросс-платформа**: мобильное приложение (Flutter) + веб (Flutter Web).
- **ИИ-анализ**: по кнопке последние 10 записей расшифровываются на клиенте, отправляются через наш прокси в OpenRouter. Не логируются.
- **Голосовые**: транскрипция через Groq Whisper (тот же принцип — прокси без логирования).

## Архитектура шифрования

Из пары `login + password` через Argon2id деривируются два независимых ключа:

- `master_key` — никогда не покидает устройство, шифрует записи (XChaCha20-Poly1305).
- `auth_key` — отправляется на сервер вместо пароля. Сервер хранит `bcrypt(auth_key)`.

Подробнее: см. [docs/crypto.md](docs/crypto.md) (TBD).
